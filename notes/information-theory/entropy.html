Entropy

Quantifying the intrinsic uncertainty, unpredictability, or “chaos” inherent in a probability distribution.

1. Definitions

While Information Content (Surprisal) measures the information gained from a single specific event, Entropy measures the average amount of information produced by a stochastic source of data.

Definition: Shannon Entropy

For a discrete random variable \(X\) with possible states \(x_1, \dots, x_n\) and probability mass function \(P(x)\), the Shannon Entropy \(H(X)\) is defined as the mathematical expectation of the surprisal \(h(X)\):

\[ H(X) = \mathbb{E}_{X \sim P}[h(X)] = -\sum_{x \in X} P(x) \log_2 P(x) \]

(Note: In the case where \(P(x) = 0\), we adopt the convention that \(0 \log_2 0 = 0\), which is consistent with the limit \(\lim_{p \to 0} p \log_2 p = 0\).)

2. Intuitions & Remarks

Expected Surprisal

Because Entropy is the Expected Information Content, it gives us a single scalar value describing the “flatness” or “chaos” of a distribution.

  • Low Entropy: The distribution is highly skewed or deterministic. The outcomes are predictable. Because you almost always know what is going to happen, observing an outcome yields very little new information on average.
  • High Entropy: The distribution is flat and uniform. The outcomes are highly unpredictable. Because you never know what is going to happen, each observation provides a lot of new information.

Imagine a heavily biased coin that lands on Heads 99% of the time. The entropy is extremely low because you are rarely surprised. But a fair coin (50/50) has maximum entropy; the outcome is a complete mystery every time.

3. Formulas & Key Properties

Key Properties of Entropy

1. Bounds

For a discrete variable with \(k\) possible states, the entropy is strictly bounded: \[ 0 \le H(X) \le \log_2 k \] * \(H(X) = 0\) if and only if the outcome is completely certain (i.e., \(P(x_i) = 1\) for some \(i\) and \(0\) for all others). * \(H(X) = \log_2 k\) if and only if \(X\) follows a Uniform Distribution (\(P(x_i) = \frac{1}{k}\) for all \(i\)). The uniform distribution represents the state of maximum uncertainty.

2. Concavity

Entropy \(H(P)\) is a strictly concave function of the probability distribution \(P\). This mathematically formalizes the idea that mixing two different probability distributions together increases the overall uncertainty.

3. Continuous Extension (Differential Entropy)

For a continuous random variable \(X\) with a probability density function \(p(x)\), the sum is replaced by an integral: \[ h(X) = -\int_{\mathcal{X}} p(x) \ln p(x) \, dx \] Warning: Unlike discrete entropy, differential entropy can be negative, and it is not invariant to reparameterization.

7. Practice Problems

Problem 1: Biased Coin Entropy

Prove that the entropy of a fair coin flip is exactly 1 bit. What happens to the theoretical entropy \(H(X)\) if the coin is heavily biased such that \(P(\text{Heads}) = 0.99\)?
Reveal Solution to Problem 1
Solution

Fair Coin: For a fair coin, \(P(\text{Heads}) = 0.5\) and \(P(\text{Tails}) = 0.5\). Using the Shannon Entropy formula: \[ \begin{aligned} H(X) &= - \sum_{x} P(x) \log_2 P(x) \\ &= - [0.5 \log_2(0.5) + 0.5 \log_2(0.5)] \\ &= - [0.5(-1) + 0.5(-1)] \\ &= 1 \text{ bit} \end{aligned} \]

Biased Coin: For a biased coin with \(P(\text{Heads}) = 0.99\) and \(P(\text{Tails}) = 0.01\): \[ \begin{aligned} H(X) &= - [0.99 \log_2(0.99) + 0.01 \log_2(0.01)] \\ &\approx - [0.99(-0.0145) + 0.01(-6.6438)] \\ &\approx 0.0143 + 0.0664 \\ &\approx 0.0807 \text{ bits} \end{aligned} \]

The entropy drops from 1 bit to roughly 0.08 bits. Because the outcome is almost certain, you gain very little information on average from observing the coin flips.

Problem 2: Maximum Entropy of a Uniform Distribution

Use the method of Lagrange multipliers to prove that for a discrete random variable \(X\) with \(k\) possible states, the entropy \(H(X)\) is maximized when the distribution is uniform, i.e., \(P(x_i) = \frac{1}{k}\) for all \(i\).
Reveal Solution to Problem 2
Solution

We want to maximize the entropy function: \[ H(P) = -\sum_{i=1}^k p_i \ln p_i \] (Note: We use the natural logarithm \(\ln\) to simplify differentiation, which only scales the result by a constant and does not change the argmax).

This maximization is subject to the probability constraint that the sum of all probabilities must equal 1: \[ \sum_{i=1}^k p_i = 1 \implies \sum_{i=1}^k p_i - 1 = 0 \]

We define the Lagrangian \(\mathcal{L}\) with a multiplier \(\lambda\): \[ \mathcal{L}(p_1, \dots, p_k, \lambda) = -\sum_{i=1}^k p_i \ln p_i + \lambda \left(\sum_{i=1}^k p_i - 1\right) \]

Take the partial derivative with respect to a specific \(p_i\) and set it to 0: \[ \begin{aligned} \frac{\partial \mathcal{L}}{\partial p_i} &= -(\ln p_i + p_i \cdot \frac{1}{p_i}) + \lambda = 0 \\ &= -\ln p_i - 1 + \lambda = 0 \end{aligned} \]

Solving for \(p_i\): \[ \ln p_i = \lambda - 1 \implies p_i = e^{\lambda - 1} \]

Notice that the solution for \(p_i\) depends only on \(\lambda\), which is a constant. This means every \(p_i\) must be exactly the same! Since there are \(k\) states and they must sum to 1: \[ p_1 = p_2 = \dots = p_k = \frac{1}{k} \]

Thus, the Uniform Distribution maximizes entropy. \(\blacksquare\)
